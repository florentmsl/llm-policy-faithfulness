Reward function used during training:
```python
def reward_fn(prev_state, state):
    """
    Representative Kangaroo-style reward:
    +1 whenever the in-game score increases (climbing up levels, punching
    fruit/bell, reaching the joey), else 0.
    """
    if state["score"] > prev_state["score"]:
        return 1.0
    return 0.0
```

```python
def play(state):
    if state["D(Ball, Paddle).y"] > 2:
        return 2  # DOWN
    if state["D(Ball, Paddle).y"] < -2:
        return 1  # UP
    return 0  # NOOP
```

Explanation:
This policy makes the paddle follow the ball's vertical position. When the ball is lower than the paddle, it moves the paddle down; when the ball is higher, it moves the paddle up; and when they are already close in height, it waits. Overall, it behaves like a simple vertical tracking controller.
